# Drop table

La sentencia DROP TABLE se utiliza para eliminar una tabla existente en una base de datos.

#### Sintaxis:

```ssh
DROP TABLE table_name;
```

#### Ejemplo:

La siguiente sentencia SQL elimina la tabla existente "Shippers":

```sql
DROP TABLE Shippers;
```
